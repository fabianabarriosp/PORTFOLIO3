## Description

## Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/fabianabarriosp/PORTFOLIO3
   ```
2. **Navigate to the project directory**:
   ```bash
   cd PORTFOLIO3
   ```
3. **Open the index.html file in your browser** to view the portfolio locally.
